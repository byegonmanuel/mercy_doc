Secure Access Workflow

Security is paramount. Our role-based access system ensures that sensitive data remains protected while enabling rapid deployment of ground teams.

1

Temporary Credentials

Admins generate one-time logins for new Coordinators and Agents. These credentials are encrypted and sent via secure internal channels (WhatsApp or SMS).

2

Mandatory Profile Update

First login triggers a forced password change and Profile Picture upload. This ensures accountability and team identification for all users.

3

Permissions Grant

The app requests Location Services for tracking and Notifications for real-time alerts. Granting these is mandatory for active duty.

4

Access Dashboard

Once authenticated, users land on a role-specific dashboard with tailored tools, data views, and action items based on their access level.

Who Uses The Platform?

A structured hierarchy ensures data security, efficient delegation, and seamless communication across the entire campaign. Each role has specific permissions and responsibilities.

The Admin

The Commanders

  • Generate login credentials for all users
  • View global analytics and reports
  • Manage resource disbursement
  • Complete polling station overview
  • Configure system settings

The Coordinator

Ward Captains

  • Manage assigned agents in their ward
  • Review ward-level statistics
  • Schedule and coordinate local events
  • Submit reports to HQ
  • Monitor agent activity and performance

The Agent

The Ground Force

  • GPS check-ins at polling stations
  • Update voter data in real-time
  • Upload field photos and reports
  • Receive notifications from coordinators
  • Access manifesto and campaign materials

Voter Database Management

A centralized, secure repository of voter information accessible to authorized team members. Agents can add, update, and categorize voter data on the ground.

Key Features

  • Real-time voter registration updates
  • Sentiment tagging (Supporter / Undecided / Opposition)
  • Contact history and interaction logs
  • Export data for offline canvassing

Data Security

All voter data is encrypted end-to-end and stored on secure cloud servers. Access is logged and monitored. Only admins can export complete datasets, while agents see filtered views relevant to their assigned areas.
